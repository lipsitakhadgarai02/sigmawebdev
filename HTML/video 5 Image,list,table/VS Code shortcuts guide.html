Shortcut to Format Code

For Windows / Linux

Shift + Alt + F

For Mac

Shift + Option + F

How it Works

This shortcut automatically:

  • Indents your code properly

  • Aligns braces {}

  • Fixes spacing

  • Makes the code clean and readable

If It Doesnt Work

Try these fixes:

  1. Set the default formatter

    • Open Command Palette Ctrl + Shift + P

    • Type:

      mathematica
      Format Document With...

    • Choose your formatter (e.g. Java, Prettier, etc.)

    • Then click Set as Default Formatter

  2. Enable Format on Save (optional)

    • Go to: File Preferences Settings

    • Search: Format On Save

    • Check Editor: Format On Save

    • Now every time you save (Ctrl + S), your code auto-formats!

Example:

Before:

java
class Hello{
public static void main(String[]args){
System.out.println("Hello");
}
}

After pressing Shift + Alt + F:

java
class Hello {
    public static void main(String[] args) {
        System.out.println("Hello");
    }
}
